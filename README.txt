TebakNamaIndonesia (Local Scores)
==================================

- Tidak ada leaderboard online.
- Login dengan nama saja (tanpa password).
- Skor disimpan secara lokal pada perangkat di localStorage per nama pemain.
- Setiap pertanyaan memiliki timer 45 detik.
- 1 jawaban benar = 2 poin. Total maksimal = 100 (50 soal).
- Untuk memainkan: upload file ke GitHub Pages atau buka index.html langsung di browser (offline).
